Docutec Softwarearen etika librea Softwarea, etika eta askatasuna: hiru hitz, hamaika balore eta makina bat arduradun. Softwaregintzaren itsasoan software librearen olatuak etika teknologikoa eta erabiltzaileon askatasuna urazaleratzen ditu. Etikoa da, elkarlana eta ezagutza partekatzea ahalbidetzen duelako. Eta librea, erabiltzailearen askatasunaren defentsa lehentasun osoko auzitzat hartzen duelako. Software librearen filosofia da teknologien munduan giza-eskubideak errespetatu eta bermatzen dituen teknologia mota bakarra. Software librea beraz, gure izaera teknologikoa etikoagoa eta askeagoa bihurtzen duen olatu erraldoia da. Eta inoiz bustiko gaitu, seguru. Baina zer da software librea? Software librea ezaugarri konkretu batzuk betetzen dituen software edo programa informatiko bat da. Programa bat, agindu multzo bat, eta horri iturburu-kodea esaten zaio. Iturburu-kodea irekia edo itxia izan daiteke. Alegia, programa garatzen duenak iturburu-kodea babesteko eta horren kontura aberasteko neurriak hartzen baditu, programa hori itxia izango da. Software librearen ezaugarri nagusia iturburu-kode irekia duela da. Programa bat hartu eta iturburu-kodea irakurtzeko aukera daukagu: hartu, irakurri, aldatu, hobetu eta berriz banatu. Egia da programa batean aldaketak egiteko gaitasuna eta ezagutza informatikoak behar direla, baina software librea hori baino gehiago ere bada: abantaila teknikoetatik haratago, software libreak balore etikoak transmititzen ditu: bizitzeko filosofia aske bat defendatzen du. Elkarlanaren etika librea Software libreak lankidetza-eredu bat definitzen du: bere garapena elkarlanean eta partekatzean oinarritutako proiektuetan banatzen da. Kasu asko eta askoetan (gehienetan) boluntarioak dira proiektu horiek aurrera ateratzen dituzten pertsonak. Eta guztiek 'komunitatea' osatzen dute. Helburu bat partekatzen dute: software etikoa garatu eta zabaltzea. Berdin dio non jaio zaren. Internetek klik bateko distantzian kokatzen gaitu: distantzia eta denbora dimentsiorik gabeko auzolan birtuala ahalbidetzen du. Software libreak komunitatearen kontzeptua bere baitan darama: lana elkarrekin egin eta mundu guztiari zabaltzea du xede. Software libreak elkarlanaren etika eta hartu eta emateko askatasuna bermatzen digu. Ezagutza partekatuaren etika librea Informazioaren eta ezagutzaren jendartean bizi garela esan ohi da. Eta egia da. Internet informazioaren autobidea da eta ezagutza modu eraginkor eta eroso batean partekatzeko bidea ematen digu. Software libreak besteek egindakotik ikasteko aukera ematen du. Ez du ezer ezkutatzen, alderantziz, ezagutza banatua sare bihurtzen du. Software libreak ezagutza partekatuaren etika eta ikasi eta irakasteko askatasuna bermatzen digu. Hizkuntzaren etika librea Software libreak eremu urriko hizkuntzak ere ezagutzen ditu. Ez du inolako mugarik ezartzen: tokian tokiko hizkuntza eta errealitateetara moldagarria da. Software libreak programak euskarara itzultzeko (lokalizazioa) eta programa Euskal Herriaren ezaugarrietara egokitzeko (internazionalizazioa) aukerak eskaintzen dizkigu. Softwaregintzaren uretan software librea euskararen arnasa da. Ito beharrean garapenera gonbidatzen gaitu. Euskara hizkuntza global eta berritzailea bihurtzen laguntzen du. Teknologia berrien munduan euskarak erronka berriak ditu eta software librea euskarari ateak irekitzen dizkion teknologia mota bakarra da. Informazioaren kapitalisten hiztegian, software libreak bakarrik definitzen du 'merkatua' euskararengan. Software libreak hizkuntzaren etika eta euskaraz bizi eta lan egiteko askatasuna bermatzen digu. Etika librerik gabeko arduradunak Software libreak demokrazia digitala bermatzen du: teknologia mundu guztiaren eskutan jartzen du. Teknologia bera demokratizatu egiten du. Administrazio publikoak teknologia herritar guztien eskutan jarri behar du, eta horretarako halabeharrezkoa da software librearen filosofiak defendatzen dituen formatu irekiak eta estandarrak erabiltzen hastea. Halaber, administrazio publikoan erabiltzen diren milaka ordenagailuetan Windows erabiltzeko egiten diren inbertsio ikaragarriak aurreztu eta teknologia etikoa eta librea erabiltzea exijitu behar dugu. Berdin hezkuntza sistemaz mintzo garenean. Hezkuntza sistemak balore etikoetan eta elkarlanerako lankidetza ereduan oinarritutako teknologien ezagutza eta erabilera sustatu eta bermatu behar du. Eta halaxe da: etikoa eta askea izanagatik, software librea ez da behar bezainbeste erabiltzen. Munduko erabiltzaileen gehiengo handi batek software jabeduna erabiltzen du. Baina software librearen olatu erraldoia hor egongo da zain: etika teknologiko librerik ez duten arduradunak bustiarazi behar ditugu orain.
